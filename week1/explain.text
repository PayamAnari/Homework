1- How do I create a hidden file or folder? How do I display it in the CLI?
Use the mkdir command to create a new folder. To make that folder hidden, add a dot (.) at the beginning of the name
mkdir .explain.text

2- How do I create multiple nested directories
We can use the -p option with the mkdir command to create a directory inside another directory.
touch explain.text
